# Linker
Linker is a Knowledge-graph converter. Visualize users' learning materials in the form of knowledge graphs and learning-friendly blogs.
# 실행방법
```
npm start
```
## GPT4 API key
```
server.cjs 와 upload/script.js 에 본인의 API KEY를 입력해주세요
```
